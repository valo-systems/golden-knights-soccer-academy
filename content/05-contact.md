# Contact, Page Content Spec

| | |
|---|---|
| **Slug** | `/contact` |
| **In nav** | Yes |
| **Priority** | Phase 1 (launch) |
| **Primary audience** | All |
| **Page goal** | Make it effortless to reach the academy for any reason. |
| **Primary CTA** | Submit enquiry form / WhatsApp click-to-chat |

---

## Section 1, Page hero

**Heading:** Get in touch
**Sub:** *Questions about joining, sponsoring, or anything else? We'd love to hear from you.*

---

## Section 2, Enquiry form

**Fields:**
- Name (required)
- Email (required)
- Phone
- Reason (dropdown: Join the academy / Sponsorship / General enquiry)
- Message (required)

**Behaviour:** routes to academy email; success + error states; spam protection (honeypot/captcha).
